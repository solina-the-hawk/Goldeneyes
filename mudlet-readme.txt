======================================================
Goldeneyes - Gold Tracking & Distribution Utility
Version: 1.0.0
Author: Solina
======================================================

Goldeneyes is a robust Mudlet ledger designed for Achaean hunting parties. It automates gold tracking, party distribution, auto-looting, and math checks for hidden quest rewards.

KEY FEATURES:
- Automated party tracking with interactive click-to-add prompts.
- Even or Fair (prorated) split strategies.
- Snapshot/Check system to find untracked gold rewards.
- Automatic data saving to protect against client crashes.
- Auto-looting to designated containers and auto-handover to party accountants.

INSTALLATION NOTE:
Ensure that "Goldeneyes-Core.lua" is saved in a folder named "Goldeneyes" inside your Mudlet Home Directory before installing this XML package. 
Path: [Mudlet Home Dir]/Goldeneyes/Goldeneyes-Core.lua

QUICK START:
Once installed, type "goldeneyes help" or "gold help" in the command line to view the full list of controls and settings.

To view your current ledger, simply type "gold".
